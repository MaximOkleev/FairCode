# Описание

Программа предназначена для сравнения исходных `C++` кодов.

Пользователь передает программе множество `.cpp` файлов, которые сохраняются в базе данных.
Основным функционалом программы является определение процента схожести каждой пары решений.

Программа позволяет выявлять схожие участки кода, что позволяет оценивать оригинальность работы, что необходимо для
соревнований.

# Основной функционал

- Программа получает `.zip` архив по прямой ссылке;
- Распаковка и сохранение файлов в базе данных;
- Поиск совпадающих решений;
- Вывод результатов сравнения пользователю;

# Как это работает

1. Программа получает `.zip` архив с запакованными `.cpp` файлами и помещает их в базу данных;
2. Обращается к базе данных для получения текста кода;
3. С помощью `DSU` строится лес совпадений;
4. Выводим все решения, которые состоят в дереве и размером превышающим два вершины.
